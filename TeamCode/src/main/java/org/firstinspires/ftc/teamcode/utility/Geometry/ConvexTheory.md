# 凸多边形的几何性质与定理
## 1. 凸多边形的定义与判定

### 1.1 定义
一个简单多边形称为**严格凸多边形**，当且仅当它的所有内角均严格小于 π（180°），并且任意三个连续顶点不共线。

等价定义：对于多边形顶点按逆时针顺序排列的序列 \(V_0, V_1, \dots, V_{n-1}\)，所有连续边段的有向转角（叉积）具有相同的严格非零符号，即  
\[
\forall i \in [0, n-1], \quad 
\mathrm{cross}( \overrightarrow{V_iV_{i+1}}, \overrightarrow{V_{i+1}V_{i+2}} ) > 0
\]  
（若为顺时针则全部 < 0）。

### 1.2 判定定理
**定理（正外角定理）**：给定按逆时针方向排列的简单多边形，若对所有 i ，边向量 \(\vec{e}_i = V_{i+1} - V_i\) 与 \(\vec{e}_{i+1} = V_{i+2} - V_{i+1}\) 的叉积同号且不包含零，则该多边形为严格凸多边形；否则若出现异号或全为零，则不是凸多边形。

**证明**：
- 内角 θ 与叉积的关系：\(\mathrm{cross}(\vec{e}_i, \vec{e}_{i+1}) = |\vec{e}_i||\vec{e}_{i+1}| \sin\theta\)。  
  对于逆时针多边形，左侧内角 θ < π 等价于 \(\sin\theta > 0\)，因此叉积为正。  
- 若所有叉积均大于零，则每个内角均小于 π，多边形严格凸。  
- 若存在叉积为负，则该处内角大于 π，多边形为凹。  
- 若所有叉积的绝对值均接近零（共线），则多边形退化为线段或点，不是严格凸多边形。

## 2. 点包含判定（点在凸多边形内）

**定理（凸多边形的半平面表达式的推论）**：设凸多边形顶点按逆时针顺序给出，则平面内一点 \(P\) 在多边形内部（含边界）的充要条件是：对于多边形的每条有向边 \(V_iV_{i+1}\)，点 \(P\) 均位于该边的左侧（或线上），即
\[
\mathrm{cross}( V_{i+1} - V_i,\; P - V_i ) \ge 0 \quad (\text{对容差 } \varepsilon \ge 0).
\]

**证明**：
- 凸多边形可以视为其所有边所在直线定义的半平面的交集：  
  \[
  \text{Polygon} = \bigcap_{i=0}^{n-1} \{ P \mid \mathrm{cross}(\overrightarrow{V_iV_{i+1}}, \overrightarrow{V_iP}) \ge 0 \}.
  \]
- 对于逆时针凸多边形，内部区域恰好是上述半平面的交。因为多边形是凸的，并且每个边都作为支撑线，取左侧半平面即可完全界定多边形区域。  
- 该性质是凸集分离定理的一个推论：若点位于多边形外，则存在一条边使得点在该边的右侧（叉积 < 0）。

## 3. 多边形包含判定
**定理（凸套定理）**：设 \(A, B\) 为两个凸多边形（凸集），则 \(B \subseteq A\) 当且仅当 \(B\) 的所有顶点都位于 \(A\) 内部（含边界）。

**证明**：
- **充分性**（顶点在内部 ⇒ 多边形在内部）：  
  凸多边形 \(B\) 是其顶点的凸包，即 \(B = \mathrm{conv}(\{B_i\})\)。  
  若每个顶点 \(B_i \in A\)，因为 \(A\) 是凸集，凸包必定包含于 \(A\)，故 \(B \subseteq A\)。
- **必要性**（多边形在内部 ⇒ 顶点在内部）：显然，若 \(B \subseteq A\)，则 \(B\) 的任何点都在 \(A\) 内，包括所有顶点。
---

## 4. 点到凸多边形的最近点

### 4.1 性质
- 若点 \(P\) 位于凸多边形内部（含边界），则最近距离为 0，最近点为其自身。
- 若点 \(P\) 位于凸多边形外部，则最近点必位于多边形的某条边上（而非内部顶点），可以通过计算 \(P\) 到每条边的最近点并取最小值获得。

### 4.2 证明
- 凸多边形是紧致凸集。根据 Hilbert 投影定理，对于凸集外的一点，存在唯一的最近点，且该点位于凸集的边界上。
- 由于凸多边形的边界由线段构成，最近点必然在某一整个边线段上（可能是内部点或端点）。通过比较点到各线段的最短距离，即可得到全局最近点。

## 5. 凸多边形相交判定

### 5.1 相交条件
**定理（交集存在的判）**：两个凸多边形 \(A\) 和 \(B\) 有交集（含边界）的充要条件是以下两种情况之一成立：
1. **顶点互含**：\(A\) 的某个顶点在 \(B\) 内（含边界），或 \(B\) 的某个顶点在 \(A\) 内；
2. **边相交**：\(A\) 的某条边与 \(B\) 的某条边相交（包括端点接触）。

### 5.2 证明
- **充分性**：若存在公共顶点或边相交，则显然存在公共点。
- **必要性**（反证）：假设交集非空但既无包含点又无边相交。任取交集内的一个点 \(x\)。由于 \(x\) 不在任何顶点，也不在互相包含的顶点，则 \(x\) 必然位于某条边或内部。可沿某方向移动至边界，得出必须穿过某条边，矛盾。严格证明可基于平面分割，此处省略。

### 5.3 线段相交判定
利用叉积的方向性进行跨立实验：
- 线段 \(AB\) 与 \(CD\) 相交（含端点）当且仅当：
  \[
  \big( \mathrm{cross}(\overrightarrow{CA}, \overrightarrow{CB}) \cdot \mathrm{cross}(\overrightarrow{DA}, \overrightarrow{DB}) \le 0 \big) \;\wedge\;
  \big( \mathrm{cross}(\overrightarrow{AC}, \overrightarrow{AD}) \cdot \mathrm{cross}(\overrightarrow{BC}, \overrightarrow{BD}) \le 0 \big)
  \]
  并且处理共线时的端点边界检查。

## 6. 凸多边形交集计算（Sutherland–Hodgman 算法）

### 6.1 理论基础
- 凸多边形可表示为一系列半平面的交集。
- 使用裁剪多边形（亦为凸）的每条边所定义的半平面，依次裁剪被裁剪多边形，结果保持凸性。
- 每次裁剪相当于将当前多边形与一个半平面求交，若当前多边形为凸，则结果仍为凸。

### 6.2 算法正确性
- 每次用裁剪边 \(\overrightarrow{CE}\) 处理当前多边形的每条边，根据两个端点在该边的内侧/外侧情况，保留内侧点或添加交点，生成新的顶点序列。
- 裁剪结果仍为凸多边形，但可能引入共线的多余顶点，需通过检测连续三点叉积是否为零来移除，得到严格凸多边形。
- 若最终顶点数 < 3，则交集为空或退化，抛出异常。

---

## 7. 旋转与平移不变性

**性质**：对凸多边形施加旋转、平移（或任意仿射变换）后，仍是凸多边形。

**证明**：旋转和平移保持直线的顺序和夹角，因此内角大小不变，仍满足凸性条件。
