# utility 工具库目录说明

`utility` 目录包含FTC机器人程序所需的各种数学工具、控制算法和数据处理类。

---

## 目录结构

| 文件/文件夹 | 功能说明 |
|-------------|----------|
| `Vector2D.java` | 二维向量类，提供向量运算、旋转、平移等操作 |
| `Vector3D.java` | 三维向量类，提供三维空间中的向量运算 |
| `ActionRunner.java` | 动作执行器，支持顺序执行多个动作 |
| `HypParams.java` | 全局超参数配置类，集中管理机器人控制系统中的所有超参数 |
| `Algebra/` | 代数工具目录，包含方程求解器和复数运算 |
| `BST/` | 弹道求解器目录，用于计算小球发射参数 |
| `filter/` | 滤波器目录，包含卡尔曼滤波、均值滤波等 |
| `Geometry/` | 几何工具目录，包含凸多边形、欧拉角、投影器等 |
| `LinearInterpolation/` | 线性插值目录，用于弹道数据插值 |
| `PID/` | PID控制器目录，包含PID、SVA、PIDSVA控制器 |
| `RK4/` | 四阶龙格-库塔法弹道模拟目录 |

---

## 核心类简介

### Vector2D
二维向量，FTC机器人坐标系基础类型。提供加减、缩放、旋转、点积、角度计算等功能。

### Vector3D
三维向量，用于三维空间计算，如相机坐标系变换。

### ActionRunner
按顺序执行多个Runnable动作，支持非阻塞式执行。

### HypParams
全局超参数配置类，使用 `@Config` 注解支持FTC Dashboard实时调参。超参数为无需拟合的常量（如工程参数）。
